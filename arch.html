1. Sub-Tick Intent Ledger

The Traffic Manager prevents physical collisions; this ledger prevents resource collisions. Native Screeps APIs update state at the end of the tick. If two creeps query an empty container simultaneously, both fire `withdraw` intents, but one fails, wasting CPU.
